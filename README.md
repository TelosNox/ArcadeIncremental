# Incremental Arcade Hall

Browserbasiertes Incremental Game. Spielspezifikation: [`DOCS/SPECIFICATION.md`](DOCS/SPECIFICATION.md).
Phasenplan: [`DOCS/IMPLEMENTATION_PLAN.md`](DOCS/IMPLEMENTATION_PLAN.md).

## Entwicklung

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

Deployment erfolgt automatisch über GitHub Actions (`.github/workflows/deploy.yml`)
bei jedem Push auf `main`.
